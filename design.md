# MERC-CMMS Enterprise Design System

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1e293b) - Professional, trustworthy, medical-grade reliability
- **Secondary**: Medical Blue (#2563eb) - Clinical precision, technology, innovation
- **Accent**: Teal (#0891b2) - Healthcare, cleanliness, modern medical systems
- **Success**: Emerald (#059669) - Compliance, safety, operational readiness
- **Warning**: Amber (#d97706) - Maintenance due, attention required
- **Error**: Red (#dc2626) - Critical alerts, compliance violations
- **Neutral Light**: Slate Gray (#f8fafc) - Clean backgrounds, sterile medical environment
- **Text**: Charcoal (#334155) - High contrast, readable, professional

### Typography
- **Display Font**: Inter Bold - Modern, clean, highly readable for medical data
- **Body Font**: Inter Regular - Excellent readability for dense technical information
- **Monospace**: JetBrains Mono - Serial numbers, technical specifications, code

### Visual Language
- **Medical Precision**: Clean lines, precise alignment, clinical sterility
- **Enterprise Trust**: Conservative use of color, professional layouts, consistent spacing
- **Data-Driven**: Information hierarchy, clear categorization, dashboard-centric
- **Compliance-First**: Visual indicators for regulatory status, audit trails

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth micro-interactions for status changes and data updates
- **ECharts.js**: Professional medical-grade data visualization
- **Pixi.js**: High-performance dashboard widgets and real-time updates
- **Splitting.js**: Text effects for section headers and important announcements
- **Typed.js**: Dynamic typing effects for dashboard statistics
- **Matter.js**: Physics-based loading animations for heavy data operations

### Effects & Animation
- **Subtle Fade-ins**: Content reveals as sections load
- **Status Pulse**: Critical alerts and overdue maintenance notifications
- **Progress Animations**: Work order completion, compliance progress bars
- **Data Counter**: Animated number counting for statistics
- **Hover States**: Gentle elevation and color shifts for interactive elements
- **Loading States**: Medical-themed loading indicators (heartbeat, pulse effects)

### Styling
- **Card-based Layout**: Clean separation of different system modules
- **Status Indicators**: Color-coded badges and icons for asset status
- **Data Tables**: Medical-record style tables with alternating row colors
- **Form Elements**: Clean, accessible inputs with validation states
- **Navigation**: Fixed sidebar with icon-based navigation
- **Dashboard Grid**: Responsive grid system for widget placement

### Header Effect
- **Medical Cross Animation**: Subtle animated medical cross in header
- **Real-time Clock**: Live time display with medical shift notation
- **Status Bar**: System health indicators and active user count
- **Notification Bell**: Animated notification system for alerts

### Responsive Design
- **Mobile-First**: Touch-friendly interface for tablet use in medical facilities
- **High DPI Support**: Crisp visuals on medical-grade displays
- **Accessibility**: WCAG 2.1 AA compliance for healthcare accessibility requirements
- **Dark Mode**: Optional dark theme for low-light medical environments
